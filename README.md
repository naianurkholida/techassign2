# Tech Assignment 2

## How to use
```
- clone this repo
git clone https://github.com/ZeitakuXIV/techassign2.git

- get your ubidots token
at top left, click your profile -> API credentials -> copy your token

- init device and variable
python try-ubidots.py (change token with yours)

- get variable id
open device tab, select your device, and you'll see variables. hover and find information (circled i).
you'll find your variable ids there

- run sensor code
change your variables, and run the code (python dht-ubi.py)

```
